# Privacy Policy for Shopee Review Scraper

**Last Updated:** February 14, 2026

This Privacy Policy describes how **Shopee Review Scraper** ("we", "us", or "our") handles your information.

## 1. Data Collection

We do **not** collect, store, or transmit any personal identification information (PII) or user data to external servers.

## 2. How it Works

The extension operates entirely locally on your device.

- **Data Processing:** All review data extraction and sanitization happens within your browser's memory.
- **Storage:** The extracted data is stored temporarily in your RAM solely for the purpose of generating the JSON file. Once the file is downloaded, the data is cleared.
- **Network Requests:** The extension intercepts network requests to Shopee's API strictly to read public review data. No data is sent back to us.

## 3. Third-Party Services

We do not use any third-party analytics or tracking tools (e.g., Google Analytics).
